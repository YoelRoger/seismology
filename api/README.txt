//////////PASOS A SEGUIR PARA PONER EN EJECUCION LA APLICACION\\\\\\\\\\

---Actuaremos sobre el terminal de Linux--
1)- Antes de hacer cualquier cosa debera crear su propio archivo .env dentro de la carpeta api, aqui utilizara de
ejemplo el archivo env-example.

2)- Ahora con el terminal de linux abierto procederemos al resto de pasos.

3)- Dar permisos de ejecucion a install.sh y boot.sh para poder ejecutarlos [utilizando chmod a-x "nombre"]

4)- Ejecutar "install.sh" en el terminal [sh install.sh]

5)- Para ejecutar la app en el entorno virtual activado ejecutar [sh boot.sh]

ATENCION: en caso de tener algun error con la ejecucion de los comandos sh install.sh y sh boot.sh ejecutarlos de la
siguiente manera [./"nombre"]
